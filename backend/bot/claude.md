# backend/bot

## Назначение

Telegram-боты на Aiogram. Обрабатывают вебхуки от Telegram.

## Файлы

### core.py — Основной бот
- Обрабатывает команду `/start`
- Регистрация пользователей и привязка реферальных кодов
- Webhook: `/api/webhook`
- Аутентификация: Telegram secret token

### recommendations.py — Бот рекомендаций
- Webhook: `/api/finder/recommendations/webhook`
- Доставляет рекомендации (лиды) пользователям с inline-клавиатурой

**Обработчики:**
- `/start` — привязка Telegram-канала для получения рекомендаций
- Кнопка **"Подходит"** — принять рекомендацию → отправить feedback в Embedding Processor
- Кнопка **"Не подходит"** — отклонить → собрать фидбек → извлечь правила через AI Core → сохранить в БД
- Кнопка **"Сгенерировать"** — placeholder

Использует FSM (Finite State Machine) для диалога сбора обратной связи при отклонении.
