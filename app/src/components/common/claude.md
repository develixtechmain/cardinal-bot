# app/src/components/common

## Назначение

Переиспользуемые UI-компоненты, не привязанные к конкретной фиче.

## Компоненты

| Директория | Описание |
|---|---|
| `Alert/` | Система модалок: ErrorBoundary, ExceptionModal, OverlayModal, SubscriptionModal, GlobalModals (оркестратор) |
| `Header/` | Шапка приложения с навигацией |
| `Loading/` | Компонент спиннера загрузки |
| `Buttons/` | WideButton (основная кнопка), LearnMoreButton |
| `Delimiter/` | Горизонтальный разделитель |
| `BottomSection/` | Нижняя секция страницы со ссылками |
| `AboutToolModal/` | Модалка с описанием инструмента |
| `ToolSelection/` | Модалка выбора инструмента: ToolSelectionModal, ToolSelector, ToolCard |
