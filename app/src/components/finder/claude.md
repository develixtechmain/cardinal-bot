# app/src/components/finder

## Назначение

Компоненты инструмента Finder — поиск лидов.

## Компоненты

| Директория | Назначение |
|---|---|
| `AboutModal/` | Модалка с описанием инструмента Finder |
| `CircleStatus/` | Круглый индикатор с количеством задач |
| `TasksList/` | Список задач Finder с toggle и удалением |
| `TasksStatisticsSection/` | Отображение статистики (всего, сегодня, среднее) |
| `TasksLearningSection/` | Секция обучающих материалов |
| `TasksLearningModal/` | Модалка с обучающим контентом |
| `BriefingExampleModal/` | Модалка с примерами ответов на брифинг |
