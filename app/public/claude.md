# app/public

## Назначение

Статические ресурсы, копируемые в билд без обработки Vite.

## Структура

| Директория | Содержимое |
|---|---|
| `assets/icons/` | Иконки приложения |
| `assets/finder/` | Изображения для инструмента Finder |
| `assets/finder/briefing/` | Изображения для онбординга |
| `assets/finder/task/` | Изображения для задач |
| `assets/contact-catcher/` | Изображения для Contact Catcher |
| `assets/referral/` | Изображения для реферальной программы |
| `assets/subscription/` | Изображения для подписки |
| `assets/subscription/about/` | Изображения для описания подписки |
| `assets/subscription/purchase/` | Изображения для покупки подписки |
